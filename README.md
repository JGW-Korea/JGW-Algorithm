- 기본적으로 JavaScript를 이용해서 알고리즘 문제를 풀지만, 가끔은 Java를 이용해서 푼 문제도 올릴 것입니다.

### Format
**{Keyword}: {출처} {유형} {난이도} {제목}**

#### Keyword
- `solve` : 풀이에 성공한 문제
- `fail` : 풀이에 실패한 문제
- `improve` : 코드를 개선해서 성공한 문제
- `docs` : readme 등 문서 수정
